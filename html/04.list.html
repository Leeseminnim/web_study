목록 관련 요소

비순서형 목록

  • 우유 100ml
  • 킹 에드워드 감자 1kg
  • 가염 버터 50g
  • 갓 간 육두구
  • 취향에 따른 소금과 후추

순서형 목록

  1. 감자를 자른다.
  2. 감자를 소금물에 끓인다.
  3. 우유,버터를 데운다.
  4. 감자의 물을 따라 내고 으깬다.
  5. 우유 혼합물에 넣고 섞는다.

정의형 목록

HTML
HTML이란, Hyper Text Markup Language의 약자로 웹 페이지의 구조를 설계할 때 사용하는 언어이다.
CSS
CSS란, Cascading Style Sheet의 약자로 요소에 접근하여 배치 및 스타일을 정의할 때 사용하는 언어이다.
